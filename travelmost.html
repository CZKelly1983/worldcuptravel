Pre-tournament projection. Distances are great-circle (haversine) between each team's confirmed base camp and each match venue, doubled for the return trip. Teams return to base between every fixture including knockout matches. For 1st and 2nd place finishers, the full bracket path through Round of 32, R16, QF, SF and Final is fixed by FIFA draw rules. For 3rd-place finishers, venues vary by which groups produce the 8 qualifying 3rd-place teams — the average is shown across all eligible bracket slots. Expected distance is probability-weighted using pre-tournament bookmaker odds (May 2026). Sources: FIFA official schedule, SportsTravel Magazine (base camps), MLS Soccer (confirmed venues), Wikipedia knockout bracket.
